# DDOS-BOMBER About


#### DDOS-BOMBER INSTALLATION AND RUN :- 


## Usage:

Run these commands to run DDOS-BOMBER

## > For Termux:

#### To use the bomber type the following commands in Termux:
```
pkg install git
```

```
pkg install python
```
```
pkg install python3
```
```
git clone https://github.com/priyans0830m/DDOS-BOMBER.git
```
```
cd DDOS-BOMBER
```
```
chmod +x ddos.py
```
```
python ddos.py

```
